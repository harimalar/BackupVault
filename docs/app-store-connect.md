## Description

BackupVault makes Mac backup simple.

Choose the folders you want to protect, choose where backups should live, and run backups whenever you want or on a schedule.

BackupVault is built for people who want straightforward local backups without cloud services, accounts, or complicated backup workflows.

### Features

- Back up folders to external drives or NAS destinations
- Keep complete dated snapshots for easy browsing and restore
- Restore everything or only the files you need
- Optional integrity verification after copy
- Local-first design with no accounts or cloud dependency
- NAS credentials stored in macOS Keychain
- Privacy-focused: no analytics, no ads, no tracking

### Good fit for

- Personal files
- Project folders
- Offline backup workflows
- Users who want simple Finder-browsable backups

## App Review notes draft

BackupVault is a local-first macOS backup utility for backing up user-selected folders to external drives or SMB/NAS destinations.

- Users explicitly choose the source folders to back up and the destination where backups are stored.
- File access is limited to locations the user selects through standard macOS file and folder permission flows.
- External drives are accessed only after the user selects the drive or backup folder.
- NAS destinations use SMB. When connecting to a network share, macOS may display its standard authentication prompt for the NAS username and password.
- Backups are stored locally on the selected destination. The app does not upload files to any cloud service.
- No account, sign-in, or subscription is required to use the app.
- No analytics SDKs, ad SDKs, tracking frameworks, or third-party data collection services are included.
- BackupVault does not access user files outside the folders the user has chosen to protect or the restore locations the user has chosen.
